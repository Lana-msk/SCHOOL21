1. Тест-кейс: Позитивная проверка авторизации
Название: Проверка успешной авторизации с валидными данными

Описание: Проверка корректной работы формы авторизации при вводе верных учетных данных.

Шаги:

а) Открыть сайт https://www.saucedemo.com

б) Ввести логин: standard_user

в) Ввести пароль: secret_sauce

г) Нажать кнопку Login.

Ожидаемый результат:

Пользователь успешно авторизуется.

Переходит на страницу с товарами.




2. Тест-кейс: Негативная проверка авторизации
Название: Проверка некорректной авторизации с различными сценариями

Описание: Проверка реакции сайта на ввод некорректных данных и других ошибок.

Шаги:

- Пустые поля:

а) Открыть сайт https://www.saucedemo.com

б) Нажать кнопку Login без ввода данных.

Ожидаемый результат: Появляется сообщение об ошибке "Epic sadface: Username is required"

- Неправильный логин:

а) Ввести логин: user.

б) Ввести пароль: secret_sauce.

в) Нажать кнопку Login.

Ожидаемый результат: Появляется сообщение об ошибке "Epic sadface: Username and password do not match any user in this service".
