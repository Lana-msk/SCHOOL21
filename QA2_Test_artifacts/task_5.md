## Проверка загрузки аватара (лимит 5 МБ)

**Что проверяем:** Форма загрузки фотографии профиля, которая принимает файлы не больше 5 МБ.

**Как проверяем:** Используем сайт **https://file.generator.teremokgames.com/**, чтобы создать 3 тестовых аватарки разного размера.

### Тест-кейсы

| № | Название проверки | Какой файл используем (размер) | Что должно произойти |
|---|---|---|---|
| 1 | **Правильная загрузка** | `4.9mb.jpg` (4.9 МБ) | Файл загружается успешно, система показывает новую фотографию. |
| 2 | **Ошибка (чуть больше)** | `5.1mb` (5.1 МБ) | Появляется сообщение об ошибке: **"Максимальный размер файла — 5 МБ"**. Файл не загружается. |
| 3 |  **Проверка границы** | `5mb` (ровно 5.0 МБ) | Результат зависит от системы: *или* ошибка, *или* успешная загрузка. Важно проверить, как написано в требованиях. |

##Проверка форматов телефонов и адресов в файле task_5_users.md:

1. По телефонам: Формат в скобочках (XXX) похож на российский, но коды внутри часто неправильные. Только номера с кодами 957, 977, 951 похожи на настоящие российские. Остальные коды (941, 944) в России не используются.

2. По адресам: Здесь есть проблема. Во всех случаях город или район не принадлежит указанному региону.

Общий итог: Данные выглядят как "сборная солянка". Для тестирования формы, которая ждёт правильные российские данные, они не подходят. Чтобы найти ошибки в программе, нужны реалистичные данные. Для этого лучше использовать другие генераторы.
