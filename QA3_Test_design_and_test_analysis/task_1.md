Класс 1: Обычные буквы

Английские буквы: маленькие и большие (a, b, c, A, B, C)
Русские буквы: маленькие и большие (а, б, в, А, Б, В)

Класс 2: Цифры

12345 (только цифры)
user123 (буквы + цифры)
007agent (цифры с нулями в начале)

Класс 3: Специальные символы

Символы: @, #, $, %, &, *, _, -
Скобки: ( ) [ ] { }
Примеры для тестирования:

user_name (с подчеркиванием)
ivan.petrov (с точкой)
test-user (с дефисом)

Класс 4: Пробелы

 (только пробел)
 username (пробелы в начале и конце)
user name (пробел между словами)
