## Сводная таблица проверок с ID

**Базовые проверки (визуал, навигация, контент, формы, интерактив):**
*   **ID 1.1:** Цвета, шрифты, изображения соответствуют бренду.
*   **ID 1.2:** Все элементы выровнены.
*   **ID 2.1:** Все ссылки корректны.
*   **ID 2.2:** Можно вернуться на главную по логотипу.
*   **ID 3.1:** Контент легко читается.
*   **ID 3.2:** Нет очевидных опечаток.
*   **ID 4.1:** Форма входа работает.
*   **ID 4.2:** Валидация данных в форме работает.
*   **ID 5.1:** Интерактивные элементы работают.
*   **ID 5.2:** Кнопки меняют вид при наведении.

**Специфические проверки для раздела «Как мы учим»:**
*   **ID N-1:** Главная мысль (уникальность обучения) понятна сразу (за 10 сек).
*   **ID N-2:** Процесс обучения описан чётко (есть этапы).
*   **ID N-3:** Есть доказательства (отзывы, проекты).
*   **ID N-4:** Есть ответы на главные вопросы абитуриента (в т.ч. про стоимость). *(Текущий статус: Fail)*
*   **ID N-5:** Есть чёткий следующий шаг (прямая кнопка «Подать заявку»). *(Текущий статус: Fail)*
